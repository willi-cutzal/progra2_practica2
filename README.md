# Práctica 2 - Programación 2

`Implementación de K-means`
---
K-Means es un algoritmo de aprendizaje no supervisado que agrupa los datos en K
grupos o clústeres, minimizando la distancia de cada punto al centroide más cercano.

Aplicaciones comunes:

- segmentación de clientes
- agrupamiento de comportamiento de usuarios
- compresión de imágenes

Objetivo de la práctica:
Implementar y comprender la técnica de agrupamiento K-Means, explorando su utilidad
en segmentación de datos sin etiquetas.
